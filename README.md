Este repositório contém o projeto **Loja de Sapatos**, uma aplicação em consola desenvolvida em Python, com suporte de base de dados **SQLite**.  
O objetivo é simular o funcionamento de uma loja online, permitindo a gestão de produtos, categorias, carrinho de compras e encomendas, tanto para o Cliente como para o Administrador.
Este projeto foi desenvolvido no âmbito da unidade curricular **Ambientes de Desenvolvimento Colaborativo**.


## 1. Funcionalidades

### Cliente
- Ver catálogo de produtos  
- Procurar produtos  
- Filtrar por preço  
- Ver detalhes de um produto  
- Ver produtos por categoria  
- Ver produtos relacionados  
- Adicionar ao carrinho  
- Remover ou alterar quantidades  
- Ver total da compra  
- Finalizar encomenda
